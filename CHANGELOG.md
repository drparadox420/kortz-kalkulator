# Changelog - Kortz Center Heist Master Matrix HUD

All notable updates and structural layout corrections for this optimization planner are tracked below.

## [v5.0-Alpha] - 2026-07-21
### Added
- Integrated an adaptive Local Average Engine powered by browser `localStorage` to automatically record successfully completed runs and dynamically overwrite text placeholders.
- Added an interactive "End Session" Gatekeeper Confirmation Prompt to let players manually choose whether to commit a run to permanent statistical history or safely discard a botched run.
- Appended a 3-State Dropdown Selector to the Loading Bay Van card handling variable spawns (Not Present / Use Baseline Average / Manual Finale Override).
- Embedded a permanent operational field guide note under the Section 2 heading outlining how to isolate exact live van payouts via intentionally failed finale runs.
- Engineered a collapsible layout configuration settings cog (`⚙️`) inside the top sticky overlay to handle mobile viewport changes seamlessly.

### Changed
- Standardized the core typography across the documentation and script arrays to favor authentic Grand Theft Auto Online thematic language (**Loot Bag** parameters replacing standard "knapsack" terminology).
- Corrected the mathematical calculation weight for all floor paintings (explain, see, duchesse, canis, orange, chief, cooked, sod, circleback, blueprints) to their true 50% Loot Bag capacity slot values, permanently eliminating the $2M optimization overflow bug.
- Re-ordered all 33 layout input fields and desktop tab-indices into a continuous, one-way vertical tracking path to establish a frictionless walkthrough routine through the mansion compound.
- Re-structured the Section 7 multiplayer card matrix so the Meteorite Fragments card drops cleanly inside a single right column stack underneath the Gemstone card rather than stretching full-width.
- Re-engineered the portrait viewport HUD to map a side-by-side horizontal row overlay split cleanly across the top margin of the phone screen, isolating the gold `#ffaa00` Buyer's Request items from the blue `#00e1ff` clock.
- Updated and verified official in-game asset spelling tags across the core grid matrix ("La Duchesse", "Het Gouden Hondje", and "Painting: Sod Off").
